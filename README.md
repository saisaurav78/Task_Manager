# Task Manager

A simple Task Manager application built with React and Node.js to manage tasks. Users can add, view, and delete tasks seamlessly.

## Features
- **Add Tasks**: Add new tasks with a name.
- **View Tasks**: Display a list of all tasks.
- **Delete Tasks**: Remove tasks from the list.

## Tech Stack
- **Frontend**: React, Axios
- **Backend**: Node.js, Express
- **Database**: MongoDB
